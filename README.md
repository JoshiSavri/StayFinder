# 🏡 StayFinder – MERN Stack Accommodation Platform

🌐 Live Demo: Coming Soon

---

## 📌 Project Overview

StayFinder is a full-stack web application inspired by Airbnb, built using the MERN stack. It allows users to explore, create, and manage accommodation listings such as apartments, rooms, and vacation homes. The platform offers a smooth and interactive user experience with authentication, image uploads, and location-based services.

---

## ✨ Key Features

* 🔐 User Authentication (Signup/Login)
* 🏠 Add, Edit, and Delete Property Listings
* 📍 Location-based Listings with Map Integration
* 🖼️ Image Upload using Cloudinary
* ⭐ Reviews and Ratings System
* 🔎 Search and Filter Listings
* 📱 Fully Responsive Design

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/JoshiSavri/StayFinder.git
cd StayFinder
```

---

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
MAPBOX_TOKEN=your_mapbox_token
CLOUDINARY_KEY=your_cloudinary_key
CLOUDINARY_SECRET=your_cloudinary_secret
SESSION_SECRET=your_secret_key
```

---

### 4️⃣ Run the Application

#### Start Backend

```bash
npm start
```

#### Start Frontend

```bash
npm start
```
